Eric Li Portfolio Website

A modern, responsive personal portfolio built with Next.js, TypeScript, and Tailwind CSS. This website showcases my background, professional experience, technical skills, and software engineering projects.

Live Website: https://ericqli.ca

⸻

Features

* Modern and responsive design
* Dark mode interface
* Smooth animations with Framer Motion
* Professional project showcase
* Experience and skills sections
* Resume download
* Contact information
* Optimized for desktop and mobile devices

⸻

Tech Stack

Frontend

* Next.js
* React
* TypeScript
* Tailwind CSS
* shadcn/ui
* Framer Motion

Deployment

* Vercel

Getting Started

Clone the repository:

git clone https://github.com/<your-username>/<repository-name>.git
cd <repository-name>

Install dependencies:

npm install

Start the development server:

npm run dev

Open http://localhost:3000 to view the website.
